# Source Documents

This folder contains 27 cleaned source documents used by the RAG pipeline. The
files are organized by EECS course number so the corpus is easier to inspect.

Each source file starts with this metadata format:

```text
Title:
Source URL:
Source Type:
Date Collected:
```

Some files use `Source URLs:` when one cleaned note combines multiple public
sources for the same course.

The original idea was professor reviews, but Rate My Professors was not reliable
to collect automatically. These public student-facing course materials and
r/uofm discussion notes still support the course-selection goal because they
include projects, prerequisites, workload signals, course structure, topics,
instructor history, and support resources. Reddit notes should be read as
student anecdotes, not official course facts.
