# Source Documents

This folder contains 67 cleaned source documents used by the RAG pipeline. The
files are organized by EECS course number so the corpus is easier to inspect.

Each source file starts with this metadata format:

```text
Title:
Source URL:
Source Type:
Date Collected:
```

Some files use `Source URLs:` when one cleaned note combines multiple public
sources for the same course.

The original idea was professor reviews, but professor reviews alone were too
subjective to be the whole project. These public student-facing course materials,
r/uofm discussion notes, and RMP aggregate notes support the course-selection
goal because they include projects, prerequisites, workload signals, course
structure, topics, instructor history, and professor sentiment. Reddit and RMP
notes should be read as student anecdotes, not official course facts.
